# Maktabati - PDF Library Organizer

Maktabati is a desktop application built with Electron and React for organizing and viewing your PDF library.

## Features

- **Organize PDF Books**: Keep your collection structured.
- **Internal PDF Viewer**: Read your books directly within the app.
- **Progress Tracking**: Automatically tracks where you left off.
- **Thinking Space**: Take notes as you read.

## Development

To run the project in development mode:

1. Navigate to the `Dev` directory:
   ```bash
   cd Dev
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Building

To build the portable executable:

1. Run the `Build.bat` script in the root directory.
2. The output will be generated in the root directory as `Library.exe`.
